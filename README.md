# AI Weather App

A modern, responsive weather application with AI-powered insights and interactive maps.

## Features

- 🌤️ Real-time weather data
- 🤖 AI-generated weather insights
- 🗺️ Interactive weather maps
- 🎤 Voice search support
- 📍 Geolocation support
- 📱 Fully responsive design
- ✨ Modern glassmorphism UI

## Deployment

### Deploy to Netlify (Drag & Drop)
1. Visit [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the entire project folder onto the page
3. Get your live URL instantly!

### Deploy via Netlify CLI
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Navigate to project folder
cd Ai-weather-app-main

# Deploy
netlify deploy --prod
```

### Deploy via Git
1. Push your code to GitHub
2. Go to [https://app.netlify.com](https://app.netlify.com)
3. Click "New site from Git"
4. Connect your repository
5. Deploy!

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## Technologies Used

- HTML5
- CSS3 (Modern Glassmorphism)
- Vanilla JavaScript
- OpenWeatherMap API
- Leaflet.js for maps
- Font Awesome icons

## API Key

The app uses OpenWeatherMap API. The current API key is included for demo purposes. For production use, get your own key at [https://openweathermap.org/api](https://openweathermap.org/api)
